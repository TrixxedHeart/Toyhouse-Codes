\# How to Use

Most templates will have an HTML and a CSS.

HTML codes can be used by anyone, however Toyhou.se requires you to pay for their premium to use CSS codes.

Most HTML codes here are designed to be able to be used without premium with the CSS as a bonus. Please consider supporting Toyhou.se though!

\## HTML Templates

To use HTML codes I recommend utilizing https://th.circlejourney.net/, its a live editor for Toyhouse that makes it significantly easier to edit these templates.



When finished editing a template in the editor, copy it into your character's page, make sure that WYSIWYG is disabled or it can mess it up!

\## CSS Templates

Assuming you have Toyhou.se premium, you can just copy these directly into the CSS section on Toyhou.se, they should not require any extra editing. (unless you want to)
